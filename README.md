# Lost and Found Management System

A web-based application built using Flask to help users report, search, and manage lost and found items efficiently.

## Features
- User-friendly interface
- Report lost items
- Report found items
- Backend logic using Flask
- Organized templates and static files

## Tech Stack
- Python (Flask)
- HTML, CSS
- Basic database integration
